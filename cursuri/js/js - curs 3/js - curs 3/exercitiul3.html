Creati o functie javascript care la onblur pe campul de text sa verifice valoarea introdusa si sa ia urmatoarele decizii:

  • daca este intre 0 si 18, sa seteze latimea div-ului id="b" ca fiind valoarea introdusa si sa-l coloreze in verde
  • daca este intre 18 si 40, sa seteze latimea div-ului id="b" ca fiind valoarea introdusa si sa-l coloreze in galben
  • daca este intre 40 si 100, sa seteze latimea div-ului id="b" ca fiind valoarea introdusa si sa-l coloreze in portocaliu
  • daca este oricare alta valoare, sa NU modifice latimea sau culoarea div-ului id="b" si afiseze in paragraful cu id="eroare" mesajul: introduceti un numar intre 0 si 100
